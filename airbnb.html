Vous souhaitez voyager grâce à Airbnb

Pour le premier partiel en deuxième année chez MyDigitalSchool, nous avons du, par groupe de 7 en trois jours, redesigner et ajouter de nouvelles fonctionnalités UI au site Airbnb. Nous nous sommes donc divisés en 3 groupes différents : Le marketing, le web design et le développement web. J'étais donc dans ce dernier pôle avec une autre personne. nous nous sommes divisés les tâches pour être plus efficaces. Je me suis occupés de la parti design du site tandis que lui à travaillé le JavaScript afin d'intégrer la fonctionnalité UI. Un bouton qui permet de créer aléatoirement un voyage (lieu, hôtel , activités). En trois jours nous n'avons fais que la page d'accueil mais nous avons remplis toutes les conditions.

Technologies utilisées :

  • Language de programmation 1 : HTML
  • Language de programmation 2 : CSS
  • Language de programmation 3 : JavaScript
